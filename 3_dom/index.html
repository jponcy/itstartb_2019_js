Reprise du TP conditions

On pourra reprendre les corrections apportées pour les TP sur les conditions.

(Rappel TP 1) Le but de ce premier TP est d'écrire quelques fonctions de bases :

  1. mult : retourne le produit des deux paramètres
  2. max : retourne le plus grand nombre entre les deux paramètres fournis
  3. abs : retourne la valeur absolue du paramètre
  4. square : retourne la valeur au carré du paramètre
  5. limit : affiche un message en fonction de la valeur du paramètre
    Moins de 18
    Affiche "Interdit aux mineurs"
    Moins de 0
    Affiche "T'es vraiment un génie !"
    Entre 18 et 110
    Bienvenue
    Plus de 110
    Bravo

Changements : Chaque fonction sera automatiquement appelée sur modification de champs de formulaire et le résultat sera affiché dans une div. Tous ces calcul seront effectués à partir des deux input ci-dessous et les résultat seront tous affiché dans le div#result.
Pour les calculs ne prennant qu'une seule valeur, on fera le calcul (et le résumé) sur les deux chiffres.
