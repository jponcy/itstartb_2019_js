Simples

A coder à la fois avec des boucles traditionnelles, et des modernes :

  1. transforme les éléments d'un tableau en leur opposé (1 => -1; -6 => 6; 0 => 0)
  2. prends une liste de noms, et affiche ces noms en liste numérotée. Exemple :
    1) Xavier
    2) Pierre
  3. affiche le produit des nombres impaires passés en paramètre
  4. affiche la somme des nombres pairs compris dans [100; 200] passés en paramètre
  5. affiche la moyenne des nombres passées en paramètre
  6. calcul le produit des chiffres provenant d'un tableau contenant des éléments de types différents
    Exemple : prod([1, '2', 3.0, 'toto', {a: 2}, [1, 2]) => 6

Factoriel

Ecrire une fonction factoriel avec le maximum d'algorithmes possible (réellement différents).

Pre-visualisation de commande

Écrire une fonction qui affiche le résumé d'une commande. Une ligne de commande est représenté par un objet donnant le nom (name), le prix (price), la quantité (quantity), la TVA (tva ; si 1 => 20%, si 2 => 8%).
Exemple :

 resumeCommand([
  {
    name: 'Mug',
    quantity: 5,
    price: 12,
    tva: 1
  },
  {
    name: 'Stylo',
    quantity: 24,
    price: 1.2,
    tva: 1
  },
  {
    name: 'Gommme',
    quantity: 2,
    price: 1.5,
    tva: 1
  },
  {
    name: 'Pâte',
    quantity: 3,
    price: 2,
    tva: 2
  }
]);

Donne :
1) Mug : a €
2) Stylo : b €
3) Gomme : c €
4) Pâte : d €

Prix Brut : e €
TVA : f €
Prix TTC : g €

N.B.: la fonction Object.keys donne un tableau contenant les clefs d'un objet. Exemple : Object.keys({a: 2, b: 3}) retourne ['a', 'b'].
